# Minesweeper

Minesweeper — классический сапер на C#, реализованный в CMD-консоли.

## Возможности

### Уровни сложности

- Легкий (10х10 клеток, коэффициент мин на поле — 0.2)
- Средний (20х20 клеток, коэффициент мин на поле — 0.3)
- Тяжелый (30х30 клеток, коэффициент мин на поле — 0.4)
- Пользовательский — игрок может сам задать размеры поля и коэффициент мин на поле

### Управление

Управление реализовано с помощью клавиш W, A, S, D (перемещение курсора), F (поставить флаг), Enter (Открыть клетку).
Выйти из игры можно нажатием клавиши Escape.

### Графика

Игровое поле почти в точности повторяет графику оригинального сапера — поле серое, цифры имеют разный цвет.

### Доп. функции

- Игровой таймер
- Счетчик шагов
